# 🚀 Autonomous Research Scout Agent (ARSA)

An agentic AI system that continuously scans research papers, GitHub repositories, and tech blogs to detect trends, generate insights, and suggest project ideas.

---

## 🧠 Overview

ARSA is designed as a real-time intelligent system that:
- Collects data from multiple sources (arXiv, GitHub, RSS feeds)
- Builds a semantic knowledge base using vector search (RAG)
- Uses an AI agent to analyze trends and generate insights
- Provides an interactive UI for exploration

---

## ⚙️ Tech Stack

- **LLM / Agent**: LangChain
- **Backend**: FastAPI
- **Vector DB**: FAISS
- **Data Processing**: Pandas, Scikit-learn
- **Ingestion**: Requests, BeautifulSoup, Feedparser
- **UI**: Streamlit
